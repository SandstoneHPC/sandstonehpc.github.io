Uploading a file

To upload one or more files from your local system:

  1. Begin by opening the Filebrowser app.
  2. Select the directory you wish to upload the files to.
  3. Click the Upload button in the directory toolbar at the top right of the page.
  4. Select the files you wish to upload to Sandstone from your local filesystem.
  5. Once each upload is completed, an indicator reading "Done" will appear next to the name of the upload.

Editing a file

To edit remote files with Sandstone:

  1. Begin by opening the Editor app.
  2. In the filetree on the left, navigate to the file you wish to open.
  3. Select the file, and then click the Edit button in the toolbar above the filetree.
  4. Font size, syntax highlighting, and other settings can be changed by opening the Settings icon in the bottom right corner of the editor.
  5. Changes can be saved by opening the dropdown menu on each open document's tab.
